# 金融市场信息面总框架

## 总目标

建立一个伞状金融市场情报框架，用来统一管理后续所有子板块要求。任何新增子板块、输出格式、信息源、评分规则、分析口径，都应先纳入本框架，再决定是否沉淀为独立 child skill。

重点覆盖：

- 美股
- 黄金
- 中国 A 股
- 加密货币
- 全球主要指数
- 央行与宏观政策
- 地缘政治大新闻
- 跨资产综合判断

核心关注资产和板块化分析要求见 `coverage-and-analysis-requirements.zh.md`。该文件定义哪些资产必须每次覆盖、哪些外围事件需要纳入、以及 A 股/美股/加密如何按板块分析。

## 核心原则

1. **事实和观点分离**
   - 事实包括来源、时间、主体、事件、数据、链接、价格反应。
   - 观点包括影响方向、传导路径、置信度、资产映射、二阶影响。

2. **先结构化，再分析**
   - 原始消息必须先进入统一 taxonomy。
   - 分析和简报应尽量基于结构化对象，而不是散乱新闻。

3. **跨资产验证**
   - 重大信息要检查股票、利率、汇率、商品、波动率、加密资产之间是否互相确认。
   - 若资产反应分歧，应明确指出分歧，而不是强行归纳。

4. **信源分层**
   - `primary`：央行、监管、交易所、统计部门、公司公告、官方声明。
   - `high`：主流通讯社、主流财经媒体、专业金融数据源。
   - `medium`：行业媒体、分析博客、研究摘要。
   - `low`：社交媒体、爆料、单一未确认消息。

5. **置信度显式化**
   - `high`：官方确认或多个高质量来源一致。
   - `medium`：可靠来源报道，但缺少一手确认。
   - `low`：传闻、社交媒体线索、逻辑推断较多。

6. **来源先分层，再分析**
   - 详细来源规划见 `source-map.zh.md`。
   - 任何重大判断都应说明来源质量。
   - 社交媒体和单一截图只能作为预警，不作为结论。

## 通用分析路径

1. 发生了什么？
2. 这是新信息，还是旧叙事的延续？
3. 直接影响哪些资产？
4. 通过什么路径传导：盈利、利率、流动性、汇率、风险偏好、监管、供应链、杠杆？
5. 市场是否已经反应？
6. 有哪些二阶影响？
7. 哪些信息能确认或推翻当前判断？

## 重要性评分

每项 1-5 分：

- 市场影响
- 新颖度
- 信源质量
- 影响广度
- 紧急程度

```text
importance = 市场影响 + 新颖度 + 信源质量 + 影响广度 + 紧急程度
```

解释：

- 21-25：最高优先级，简报头条。
- 16-20：重要，进入主简报。
- 11-15：相关，若能连接主题则纳入。
- 5-10：背景或观察名单。

## 分市场核心问题

### 美股

- 影响指数、行业、风格因子，还是个股？
- 改变盈利预期、折现率、流动性，还是风险偏好？
- 是否引发板块轮动？

### 黄金

- 金价是在交易实际利率、美元、通胀，还是避险？
- 是否被美债收益率、美元指数、ETF 流、央行购金确认？
- 地缘因素是短线避险还是结构性买盘？

### 中国 A 股

- 属于政策宽松、流动性、产业政策、地产链，还是监管风险？
- 哪些行业有一阶影响？
- 国内资金和外资定价是否一致？

### 加密货币

- 属于宏观流动性、监管、ETF 资金流、链上结构，还是杠杆出清？
- BTC、ETH、山寨币风险偏好是否分化？
- 是否与 Nasdaq、美元流动性、实际利率相关？

### 央行与宏观

- 是否改变政策反应函数？
- 是否改变市场对利率路径的定价？
- 传导路径是利率、流动性、汇率，还是预期管理？

### 地缘政治

- 是标题风险、供应冲击、需求冲击、制裁风险，还是军事升级？
- 哪些资产一阶暴露？
- 升级和缓和路径是什么？
