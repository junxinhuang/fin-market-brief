# 金融市场跟踪日报

静态 HTML 日报发布目录。

## 发布结构

- `index.html`：日报首页和归档入口。
- `reports/daily/`：每日完整 HTML 报告。
- `reports/templates/`：日报 HTML 模板。
- `skills/`：日报生成框架和规则。

## GitHub Pages

建议在 GitHub 仓库 Settings -> Pages 中选择：

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`

启用后，飞书日报推送可使用 GitHub Pages 链接。
